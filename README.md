# 线圈生成器

[English](./README.en.md)

## 功能概述

线圈生成器是一款专为 PCB 设计人员开发的工具，旨在根据用户输入的参数快速生成适用于 NFC 卡片、PCB 电机和无线充电等项目的 PCB 线圈。当前支持以下线圈类型：

-   螺旋线圈
-   方形线圈

用户可以通过以下参数配置线圈：

-   线圈圈数
-   内半径
-   轨道间距
-   轨道宽度
-   线圈中心的X、Y坐标

## 用户界面

用户可以通过顶部菜单栏访问以下功能：

-   PCB页面 > 线圈 > 线圈生成器
-   PCB页面 > 线圈 > 关于
